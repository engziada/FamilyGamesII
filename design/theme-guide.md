# Family Games UI Theme Guide

## Color Palette
- **Primary (Fun Red):** `#FF6B6B`
- **Secondary (Cool Teal):** `#4ECDC4`
- **Accent (Sunny Yellow):** `#FFE66D`
- **Background Main:** `#F7FFF7`
- **Surface:** `#FFFFFF`
- **Text Dark:** `#2F2F2F`
- **Text Light:** `#5F5F5F`
- **Success:** `#6BCB77`
- **Danger:** `#FF6B6B`
- **Purple Highlight:** `#BC7AF9`

## Typography
- Display / Headings: `Lemonada`
- Body: `Changa`

## Core Components
- `.btn`, `.btn-primary`, `.btn-secondary`, `.btn-outline`
- `.card` with irregular border radius and dashed outline using CSS variables
- `.modal` + `.modal-content` for overlay dialogs
- `.input-group`, `.input-control`
- `.game-container`, `.game-board`, `.sidebar`, `.display-area`

Use `style.css` for full declarations.
