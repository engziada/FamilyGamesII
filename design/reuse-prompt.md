# Family Games UI Prompt

Use this prompt when you want to recreate the "Family Games" UI vibe in a different project:

```text
Create a playful RTL-friendly interface inspired by Egyptian family party games. Use the following guardrails:
1. Color palette: primary #FF6B6B, secondary #4ECDC4, accent #FFE66D, background #F7FFF7, text #2F2F2F.
2. Typography: Lemonada for headings, Changa for body. Emphasize chunky rounded Arabic lettering.
3. Components: irregular "squircle" cards with thick borders and dashed shadows, pill-shaped primary buttons, outlined accent buttons, and blurred glassy modals.
4. Layout: responsive grid cards on landing page, centered modals with full-viewport blur, and two-column game board with sidebar that collapses on tablets.
5. Inputs: rounded pill text fields with thick black outlines, 1.5rem padding, and animate-in interactions.
6. Atmosphere: soft dotted background, floating color blobs, subtle bounce or jelly animations.
Ensure all spacing uses clamp() for responsiveness and keep interactions touch-friendly.
```
